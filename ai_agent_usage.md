# OrbiRetail AI Agent 使用说明

Batch 42 在工作台中新增 AI Agent 智能助手。当前版本采用内置规则和产品知识库，不依赖外部大模型 API，因此部署到 Streamlit Cloud 时无需配置 OpenAI Key。

## 能做什么
- 根据用户自然语言需求推荐模板。
- 说明上传字段和文件格式。
- 指导用户下载问题清单和报告包。
- 作为客服入口，提示用户如何提供问题信息。
- 对未来高校/实习/就业等场景给出字段准备建议。

## 后续可升级
正式商业版可以接入大模型 API，将 AI Agent 升级为：
- 自动生成字段映射建议。
- 自动解释异常问题。
- 自动生成经营摘要。
- 自动生成客服回复和售后优化建议。
- 自动生成 SQL/API 调用和报告配置。
