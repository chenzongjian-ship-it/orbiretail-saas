# Batch 43 AI 作业批改功能说明

## 支持文件

- Word：.docx
- PDF：.pdf
- 文本：.txt
- ZIP 作业包：.zip，内部可包含 .docx / .pdf / .txt

## 功能

- 根据作业类型与评分细则进行辅助评分
- 生成分数、等级和批改意见
- 检测作业之间的疑似雷同/抄袭风险
- 记录学生历史表现
- 对连续高分学生给出表扬标注
- 对连续低分学生给出警告标注
- 每次生成清晰的 Excel 批改数据分析表

## 当前边界

当前为规则增强型 AI 批改 MVP，适合初筛、统计、辅助反馈和批量梳理。正式课程成绩建议教师复核。后续可接入大模型 API，实现更强的语义理解、评分细则解析和深度评语生成。

## 上传容量

已提供 `.streamlit/config.toml`，将 `maxUploadSize` 设置为 500MB。Streamlit Cloud 的实际可用容量仍可能受到平台资源影响。正式商业版建议使用对象存储，例如 S3、OSS 或 Supabase Storage。
