# OrbiRetail Streamlit SaaS MVP

## 本地运行

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

浏览器打开：

```text
http://localhost:8501
```

## 部署到 Streamlit Community Cloud

1. 把本目录上传到 GitHub 仓库。
2. 登录 Streamlit Community Cloud。
3. Create app。
4. Repository 选择你的仓库。
5. Main file path 填 `app.py`。
6. Deploy。
