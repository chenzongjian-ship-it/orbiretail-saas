# 支付系统核查清单

## 资金去向

真实接入支付后，用户资金进入支付平台绑定的商户账户：

- 微信支付：你的微信支付商户号对应结算账户
- 支付宝：你的支付宝商家账户
- Stripe：你的 Stripe 账户绑定银行账户
- 企业转账：你的对公或指定收款账户

## 系统必须记录

- 内部订单号
- 用户邮箱
- 企业 ID
- 套餐
- 金额
- 支付平台
- 支付状态
- 平台交易号
- 回调时间
- 管理员确认记录

## 避免财务纠纷

1. 不允许仅凭前端按钮判定付款成功。
2. 必须以平台回调 + 验签结果为准。
3. 企业转账必须录入转账流水号。
4. 管理员后台需支持导出付款记录。
5. 会员开通、续费、退款都要留痕。
