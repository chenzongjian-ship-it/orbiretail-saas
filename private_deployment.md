# OrbiRetail 企业私有化部署说明（Batch 40）

适合企业客户在内网部署，处理门店销售、渠道结算、财务对账、工资考勤等敏感数据。

## 推荐架构

- Nginx / HTTPS
- Streamlit 或独立 Web 前端
- FastAPI API 服务
- PostgreSQL 数据库
- 对象存储或内网文件存储
- 定时备份与审计日志

## 私有化部署建议流程

1. 准备企业内网服务器。
2. 安装 Docker / Docker Compose。
3. 配置 PostgreSQL 数据库。
4. 导入业务模板和字段字典。
5. 配置管理员账号。
6. 测试注册、上传、分析、报告下载。
7. 配置企业内部域名和 HTTPS。
