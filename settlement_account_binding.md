# 结算账户绑定说明

我无法代你在支付平台绑定结算账户，因为这需要你本人或企业在支付平台完成实名/企业认证、商户号申请、银行账户绑定和平台审核。

## 微信支付

你需要在微信支付商户平台完成：

- 商户号申请
- 主体认证
- 结算银行账户绑定
- API v3 密钥配置
- 回调地址配置

## 支付宝

你需要在支付宝开放平台或商家中心完成：

- 企业/个人商家认证
- 应用创建
- 支付产品开通
- 支付宝公钥 / 应用私钥配置
- 结算账户配置

## Stripe

你需要在 Stripe Dashboard 完成：

- 企业/个人身份认证
- 银行账户绑定
- Webhook endpoint 配置
- Endpoint secret 配置

## Aurevia 后台记录什么

Aurevia 后台只记录：

- 支付平台
- 商户号 / 账号 ID
- 结算主体名称
- 结算账户尾号 / 备注
- 绑定状态
- 管理员备注

不要在 Aurevia 或 GitHub 中保存完整银行卡号、私钥、API Key。
